## 🧪 About My Work

I specialize in **API test automation**, **responsive UI testing**, and **database validation** across multiple environments. I enjoy working close to the development process — reviewing requirements, writing test plans, and making sure nothing slips through.

- 🔬 Building BDD automation frameworks with **Playwright + Cucumber + TypeScript**
- 🔁 Comparing Legacy vs Core API responses during complex migration projects
- 📊 Running performance benchmarks and generating detailed QA reports
- 🗄️ Validating database state with **SQL Server** queries during migrations
- 🚀 Integrating test pipelines into **Azure DevOps** CI/CD workflows
- 📱 Mobile & responsive testing with **BrowserStack** and **Responsive Viewer**
